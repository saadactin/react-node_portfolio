# Windows Setup Guide

## Install Node.js
- Download the latest LTS installer from [nodejs.org](https://nodejs.org/en/download).
- Run the installer and select the options to install both `Node.js` and `npm`.
- Accept the license, keep the default installation path, and finish the setup.
- Re-open PowerShell and confirm the install with `node -v` and `npm -v`.

## Run This Project
- In PowerShell, switch to the project folder, for example:
  - `cd C:\path\to\Personal-Website-Template-main`
- Install dependencies: `npm install`
- Start the development server: `npm start`
- The site opens at `http://localhost:3000`; changes in `src/` hot-reload.
